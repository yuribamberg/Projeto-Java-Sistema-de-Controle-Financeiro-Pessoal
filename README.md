# Sistema de Controle Financeiro Pessoal

Projeto desenvolvido em Java com execução via terminal, com o objetivo de simular um sistema de controle financeiro pessoal. A aplicação permite a navegação por um menu principal e servirá como base para a implementação de funcionalidades como cadastro de receitas, despesas, relatórios e organização por categorias nas próximas etapas.

## Funcionalidades da Sprint 1

* Menu principal executado via terminal
* Estrutura inicial do sistema organizada
* Navegação entre opções utilizando switch/case
* Loop de execução do menu com do-while
* Opções do sistema ainda em fase inicial de desenvolvimento

## Como executar o projeto

1. Abra o projeto em uma IDE Java (VS Code, IntelliJ ou similar)
2. Localize a classe principal (App.java ou Main.java)
3. Execute a classe normalmente
